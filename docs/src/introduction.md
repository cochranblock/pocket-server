# pocket-server

Compiled Rust web server for Android. No cloud, no monthly fees. Kiosk dashboard.

pocket-server runs on Android (or any machine) and serves static sites from local storage. It provides a live kiosk dashboard showing uptime, requests, bytes served, power draw, and estimated monthly cost — all from your phone. Part of the CochranBlock zero-cloud architecture.
